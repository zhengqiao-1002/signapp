关于星语桥

系统简介

星语桥是一个基于人工智能的手语学习平台，旨在帮助用户更好地学习和理解手语。通过先进的计算机视觉技术，我们能够实时识别用户的手语动作，并提供即时反馈。

核心功能

手语识别

通过摄像头实时识别用户的手语动作，支持多种常用手语词汇的识别。

语音转换

支持语音输入转换为文字，方便听障用户理解他人的语音内容。

手语参考库

提供丰富的手语视频资料，按类别整理，方便用户学习和查询。

个性化学习

根据用户的学习历史和使用频率，自动收藏常用手语，打造个性化学习体验。

技术栈

前端技术

  • HTML5 / CSS3
  • JavaScript (ES6+)
  • WebRTC (视频处理)
  • MediaRecorder API (语音录制)

后端技术

  • Python
  • Flask 框架
  • SQLAlchemy ORM
  • JWT 认证

AI 技术

  • TensorFlow / MediaPipe
  • 计算机视觉
  • 语音识别
  • 深度学习模型
